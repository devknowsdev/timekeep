# TimeKeep

A local time tracker for work, practice and exercise. Runs fully offline as an installed app.

## Setup (one time only)

**Mac / Linux**
1. Open Terminal in this folder
2. Run: `bash start.sh`
3. Open `http://localhost:8765` in Chrome or Edge
4. Click the install icon in the address bar (or browser menu → "Install TimeKeep")
5. Done — close the terminal. The app now lives on your desktop/dock.

**Windows**
1. Double-click `start.bat`
2. Open `http://localhost:8765` in Chrome or Edge
3. Click the install icon in the address bar → "Install TimeKeep"
4. Done — close the window. The app now lives in your Start menu.

## After install

- The app opens like any desktop app — no browser chrome, no server needed
- All data is stored locally in your browser's storage (never leaves your machine)
- Works 100% offline

## Keyboard shortcut

Press **§** anywhere in the app to start or stop the active timer.

## Browser compatibility

Chrome or Edge recommended for PWA install. Firefox supports the app but not the install-to-desktop feature.

---

*All data is stored in `localStorage` in your browser. To back up, use the browser DevTools → Application → Local Storage.*
